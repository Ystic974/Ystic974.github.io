Librairie Java SpringBoot pour la génération de PDF

Description détaillée

Lors de mon stage chez OKTEO, j'ai pus faire mes premiers pas sur le framework Spring Boot sur une base de Java.
Le sujet de mon stage était donc de réaliser une librairie permettant de générer des PDF, côté Back-end puis de les renvoyer à la partie front-end pour pouvoir les télécharger par exemple.

J'ai donc réalisé cette librairie sur ce framework en utilisant le moteur de template FreeMarker qui me permet, à partir d'un template HTML créé, de le remplir avec les données ou images souhaités en intégrant des graphique via Chart.js ou ECharts.

Une fois le fichier HTML rempli et généré, je l'envoie à une instance de Gotenberg, hébergé sur un serveur interne, pour obtenir un PDF personnalisé avec la possibilité d'y ajouter une page de garde ainsi qu'une en-tête, un pied de page et une pagination.

Pour faciliter le développeur à l'utilisation de ma librairie, j'y ai ajouté d'inititive un outil de développement. Son but est d'éviter que le développeur se perde dans le code d'un projet à la recherche de chaque méthodes qui créent un PDF. Grâce à une annotation à ajouter à une méthode de ce genre, cette dernière apparait sur l'outil de développement et le développeur peut donc tester sa méthode directement sur cette outil en remplissant ses champs et en ayant la possibilité de prévisualiser le futur PDF ou de directement de télécharger.

Videogames Manager

Description détaillée

Application réalisée pendant les cours de développement mobile en 3ème et dernière année de BUT Informatique. J'ai pu donc réalisé, avec un de mes collègue, une application mobile permettant à l'utilisateur de gérer et découvrir des jeux vidéos.

L'utilisateur peut donc découvrir le jeu actuellement à la une ainsi que rechercher un jeu qui l'intéresse par son nom via un écran de recherche et de recommendation. Enfin, un espage "profil" permet à l'utilisateur de gérer sa liste de jeu et sa liste de souhaits qu'il peut alimenter via l'écran de détail d'un jeu.

L'application a d'abord été réalisé en Jetpack Compose, basé sur du Kotlin avec une base de données locale "Room" et la librairie "Riverpod". Elle a ensuite été refaite en Flutter.
